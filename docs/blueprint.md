# **App Name**: MD Trader's Journal

## Core Features:

- Startup Animation: Provide a startup animation that incorporates the company logo, name, and inspiring trading quotes.
- New Trade Input: Enable trade data input, edit, and storage with an 8-step, step-by-step panel with full validation. 
- Local Data Audit Tool: Local data auditing to detect discrepancies between original inputs and automatically-calculated fields, using AI as a tool for complex calculations and anomaly detection, offering suggestions to resolve differences. 
- Real-Time Trade Updates: Automatic update for calculation to real time by any input/changes on trade and show history with user account. Completely offline Functionality to make changes
- Quick Market Hub: Users can customize URLs and default parameters for tools like TradingView to automate analyses
- Action History: Implement Undo and Redo buttons and logic for all actionable inputs throughout the application, preserving action history for each user.
- Theme Support: Offer both Dark and Light theme settings that maintains a visually consistent style and glassmorphic treatment of the UI.

## Style Guidelines:

- Primary color: A saturated sky blue (#4680FF), suggestive of both dawn and high-tech data visualization.
- Background color: Very light blue-gray (#F0F4F8).
- Accent color: Saturated orange (#E48900) for clear calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif with warmth and readability. 
- Code font: 'Source Code Pro' for displaying code snippets. 
- Simple, clear icons to match data or category using icon librarie
- Glowing border when User interact or click. Slight zoom-in when Hovering, to make it professional view
- Employ a Glassmorphic design approach, using translucent, blurred backgrounds with gradients for panels and UI containers.
